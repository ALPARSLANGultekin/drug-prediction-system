# Drug Prediction System 💊

This repository contains a machine learning project that predicts the most appropriate medication for a patient based on their medical and demographic data.

## 📋 Project Overview
Using a dataset from Kaggle, this project splits the data into training and testing sets and applies machine learning techniques to predict drug types based on:
* Age
* Gender
* Blood Pressure Levels
* Cholesterol Levels
* Na-to-K Ratio

## ✨ Key Features
* **Custom Data Visualization:** Includes a specific function to generate enhanced, aesthetically pleasing pie charts for data distribution.
* **Discrete Data Detection:** Features a dedicated function to identify and analyze discrete data/outliers effectively.
* **Plug and Play:** The Jupyter Notebook is fully self-contained and ready to execute.

## 🚀 How to Run
Running the project is straightforward. No special configuration or extra steps are required:
1. Clone this repository or download the `.ipynb` file.
2. Open the `190707037_Gültekin_ALPARSLAN.ipynb` file in Jupyter Notebook, Google Colab, or VS Code.
3. Run the cells sequentially from top to bottom.
